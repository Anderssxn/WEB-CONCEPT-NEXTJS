
# GONET Web Concept
This is a web design concept built for internet services providers.
## Tech Stack

Node Version 20.9.0

**Framework** NextJS 14.0.1 

**UI/UX:** Shadcn/ui, Tailwindcss







## Getting Started
First, install all dependencies:
```bash
npm install
```
Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result
